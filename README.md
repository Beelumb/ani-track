A modern, responsive web application for discovering and tracking anime. Users can search the Jikan API database, view detailed information, and log in to save anime to their personal lists (Watching, Completed, Plan to Watch, etc.).

## ‚ú® Features

-   **Authentication:** Secure login and signup powered by Supabase Auth.
-   **Anime Discovery:** Search and filter anime using the Jikan API (MyAnimeList data).
-   **Personalized Lists:** Add anime to custom statuses: *Watching, Completed, On Hold, Dropped, Plan to Watch*.
-   **Responsive UI:** A dark-themed, mobile-friendly interface built with Tailwind CSS.
-   **Data Persistence:** User data and lists are stored securely in a PostgreSQL database via Supabase.
